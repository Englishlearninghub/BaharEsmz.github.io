# English Self-Study Curriculum (A1 to C1)

Free, structured self-study English material — built around a 10-grade curriculum that takes learners from complete beginner (A1) to advanced (C1).


## About this project

This repository holds 10 grades, each with 5 units, covering grammar, vocabulary, and theme-based content mapped to the CEFR framework.

| Grade | CEFR Level | Focus |
|-------|-----------|-------|
| 1  | A1 | Beginner |
| 2  | A1–A2 | Elementary |
| 3  | A2 | Pre-Intermediate |
| 4  | A2–B1 | Intermediate |
| 5  | B1 | Intermediate |
| 6  | B1–B2 | Upper Intermediate |
| 7  | B2 | Upper Intermediate |
| 8  | B2+ | Advanced |
| 9  | C1 | Advanced |
| 10 | C1 | Proficient |


## What's inside

- 📘 10 grades × 5 units — full grammar + vocabulary progression from A1 to C1
- 🎯 Each unit tied to a clear theme (e.g. Food, Travel, Crime & Punishment, Media Literacy)
- 🔍 A searchable/filterable curriculum reference (see `Curriculum.html`)
- 🌱 Designed for self-paced, independent study

## How to use

1. Open `Curriculum.html` to browse the full curriculum — filter by level or search by grammar point, vocabulary, or theme.
2. Find your current level (start at Grade 1 if you're new to English, or jump to the grade matching your level).
3. Work through units in order — each grade builds on grammar and vocabulary from the last.
4. Move to the next grade once you're comfortable with the material.
